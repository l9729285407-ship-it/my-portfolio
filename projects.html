🏠 House Price Prediction (Linear Regression)

Full working implementation using Python.

import pandas as pd
from sklearn.linear_model import LinearRegression

# Training data
df = pd.DataFrame({'sqft': [1000, 2000, 3000], 'price': [300, 500, 700]})
model = LinearRegression().fit(df[['sqft']], df['price'])

# Predict price for 1500 sqft
print(model.predict([[1500]]))

📷 Face Detection (OpenCV)

Source for real-time video processing.

import cv2
face_cascade = cv2.CascadeClassifier('haarcascade_default.xml')
cap = cv2.VideoCapture(0)

while True:
    _, img = cap.read()
    faces = face_cascade.detectMultiScale(img, 1.1, 4)
    for (x, y, w, h) in faces:
        cv2.rectangle(img, (x,y), (x+w, y+h), (0,255,0), 2)
    cv2.imshow('FaceDet', img)
    if cv2.waitKey(1) == ord('q'): break
